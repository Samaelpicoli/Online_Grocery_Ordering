Atividade realizada com base no exercício 'Online Grocery Ordering' proposto no site: https://community.automationanywhere.com/developer-challenges-85011/challenge-pages-85136

Bibliotecas utilizadas no processo: Pandas, Selenium 4, os, getpass e time.

Para rodar o processo instalar as bibliotecas e executar o arquivo main.py, as funções do processo estão no arquivo processo.py e como foi feito para capturar o download do arquivo necessário para execução esta no arquivo pega_arquivo.py

pip install pandas
pip install selenium
pip install webdriver-manager
